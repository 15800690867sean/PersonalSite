# Sean Tang Personal Site

The personal website of Sean Tang, displaying the resume, self-introduction and contact.

# Run Project
### 1. Clone the project
```shell
git clone ssh/https link
```

### 2. Run the project
```shell
npm i --save --legacy-peer-deps
npm start
```

### 3. Build
```shell
npm run build
```

### 4. Deploy
```shell
npm run deploy
```
