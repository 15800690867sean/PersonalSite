# Sean Tang Personal Site

The personal website of Sean Tang, displaying the resume, self-introduction and contact.

# Run Project
### 1. Clone the project


### 2. Run the project
```shell
npm i --save --legacy-peer-deps
npm start
```

### 3. Build
```shell
npm run build
```

### 4. Deploy
```shell
npm run deploy
```
more see: https://javascript.plainenglish.io/deploying-any-app-to-github-pages-1e8e946bf890
